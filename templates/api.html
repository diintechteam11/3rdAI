Endpoint Reference

POST /upload-video

Accepts multipart/form-data containing the video file and selected triggers.

{
  "video_file": "binary data",
  "triggers": "ANPR,Helmet"
}
GET /video-result/{task_id}

Provides polling status, output video URL, and detection history.

Example Integration (cURL)

curl -X POST http://localhost:8000/upload-video \
     -F "video_file=@/path/to/video.mp4" \
     -F "triggers=Number Plate Detection,Helmet Detection"
