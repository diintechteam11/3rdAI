API Reference v1.2

Camera Recording System

Professional grade API for camera registry, scheduled recording, and AI analysis integration.

Base URL /api/v1

Authentication

The API uses Bearer token or X-API-Key authentication.

Bearer Format

Authorization: Bearer sk-8a7...

Custom Header

X-API-Key: sk-8a7...
GET

/cameras

Returns paginated list of cameras with searching, filtering (status, location, brand) and sorting.

GET /api/v1/cameras?search=Gateway&status=active&sort=latest&page=1&page_size=20
POST

/cameras/{id}/recording/start

Initiates manual recording. Returns session_id.

Body Fields

  • initiated_by (Required) - User ID
  • note (Optional) - Description
POST

/cameras/{id}/analysis/start

Triggers AI analysis and automatically starts a recording session tagged with source='analysis'.

POST

/cameras/{id}/analysis/stop

Completes analysis and stops current capture. Links performance metadata to the recording.
