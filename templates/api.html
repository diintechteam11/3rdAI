API Reference

Camera Recording System

Comprehensive API for managing video surveillance, automated recordings, and AI-driven violation detection events.

Base URL /api/v1
Live Playground

Authentication

The 3rdAI API uses Bearer token authentication. Every request must include the Authorization header with your secure API key.

Bearer Format

Authorization: Bearer sk-8a7c...

Custom Header Format

X-API-Key: sk-8a7c...
sk-8a7c2b9f0d1e2f3a...
GET

/cameras

Retrieve a paginated list of all registered cameras. Supports search, status filters, and sorting by creation time.

Query Parameters

search string

Search by camera name or location.

status string

active | inactive | recording

Response Structure

{
  "cameras": [
    {
      "id": "c1a2b3c4...",
      "name": "Entrance Gate 01",
      "status": "active",
      "location": "Warehouse A",
      "is_recording": false,
      "created_at": "2025-03-30T10:00:00Z"
    }
  ],
  "total": 45,
  "page": 1,
  "page_size": 20
}
POST

/cameras/{id}/recording/start

Initiates a manual recording session for a specific camera. Returns a unique session_id.

Request Body (Multipart)

initiated_by required
note optional
// Sample Python Request
requests.post(f"/api/v1/cameras/{id}/recording/start", data={"initiated_by": "Operator_02"})
POST

/analysis/start

Specially designed for AI engine integration. Automatically starts a recording and marks the source as analysis. Atomically links the capture to the AI event.

Crucial for Compliance

This endpoint creates an AnalysisSession record. The resulting video session will have the analysis_session_id attached for full audit traceability of AI-triggered events. 

{
  "session_id": "rec_8a7...",
  "analysis_session_id": "an_2b3...",
  "status": "capturing",
  "analysis_type": "object_detection",
  "capture_started_at": "2025-03-30T10:00:00Z"
}

Error Conventions

409 Conflict

Returned when starting a recording on a camera that is already active.

Standard JSON Error
{ "error": true, "code": "RESOURCE_NOT_FOUND", "message": "Camera not found." }
